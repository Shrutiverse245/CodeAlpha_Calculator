# Premium Minimalist Calculator 📱

A production-ready, highly responsive web-based calculator designed with a professional, minimal aesthetic. Built from scratch using modern Semantic HTML5, CSS Grid/Flexbox layouts, and Vanilla JavaScript. 

This project was developed as part of the *CodeAlpha Frontend Development Internship* (Task 2).

## ✨ Features

* *Minimal Professional Aesthetic:* Clean UI leveraging light drop shadows, a crisp layout, and fluid modern interactions.
* *Core Math Operations:* Seamlessly handles addition, subtraction, multiplication, and division.
* *Precision Handling:* Fixed floating-point calculation bugs (e.g., 0.1 + 0.2 outputs exactly 0.3 instead of trailing decimal numbers like 0.30000000004).
* *Robust Logic:* Safe handling for division by zero (displays a clean Error message instead of breaking the app).
* *Fully Responsive Design:* Adapts smoothly across all layouts—from high-resolution monitors to compact mobile phone screens.
* *Keyboard Support:* Full capability to operate the calculator using your keyboard numeric pad or top-row keys (Enter for =, Backspace for DEL, Escape for AC).

## 🛠️ Technologies Used

* *HTML5:* Structured semantic markup for accessibility and screen reader friendliness.
* *CSS3:* Advanced layouts using CSS Grid, responsive design practices, and customized active button states.
* *JavaScript (ES6+):* Clean state management, precise calculation algorithms, and event listening for input handling.
